Usage Guidelines
When to use Emerald Green (
#10B981):
Primary CTAs and important buttons
Data insights and positive metrics
Active navigation items
Success states
Hover states for interactive elements
Glowing data points
Growth indicators
When to use Burgundy (
#8B2E3C):
Secondary accent for premium features
Important warnings or critical data
Special badges or labels
Alternative CTA buttons
Executive-level content highlights
When to use Slate backgrounds:
#0F172A - Body background
#1E293B - Cards and panels
#334155 - Nested components
#475569 - Highest elevation elements
